Command-Line Toolkit
====================

This toolkit includes a set of tools that can be run from the Windows command-line to simplify and speed up common tasks.


Command List
-----------------------

+ tail
+ head
+ grep
+ encode
+ checksum
+ base64
+ count
+ find
+ env
+ split
+ join


Environment Requirement
-----------------------

Microsoft .NET Framework 4
